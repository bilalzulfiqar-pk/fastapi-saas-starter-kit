# Start Here

This repo is meant to be cloned, renamed, and extended into a real SaaS product.

Start with:

1. Run the foundation locally with `docker compose up --build`
2. Verify the starter with the checks listed in the root `README.md`
3. Decide which optional post-MVP modules you actually need
4. Rename the project, branding, cookie names, and app URLs as needed
5. Add product-specific backend and dashboard modules only after the shared foundation feels stable
